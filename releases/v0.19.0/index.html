Usage

Upload the release tarball from the command line:

bosh upload-release --name="ssoca" --version="0.19.0" \
  "https://s3-external-1.amazonaws.com/dpb587-ssoca-bosh-release-us-east-1/artifacts/release/0.19.0/ssoca-0.19.0.tgz" \
  --sha1="sha256:3b171fb2709b3f796ed61d47050f5a83e7feb7ba62daa960e0f17be2f111adcb"

Or reference the release from a manifest:

releases:
- name: "ssoca"
  version: "0.19.0"
  url: "https://s3-external-1.amazonaws.com/dpb587-ssoca-bosh-release-us-east-1/artifacts/release/0.19.0/ssoca-0.19.0.tgz"
  sha1: "sha256:3b171fb2709b3f796ed61d47050f5a83e7feb7ba62daa960e0f17be2f111adcb"
